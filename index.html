GIS Dataset Brasil

Geographic Information Systems (GIS) Dataset Brasil. Coleções de arquivos com dados geoespaciais em forma de vetor.

Conjunto de dados

  • UF
  • Mesorregião
  • Microrregião
  • Municípios

Formatos disponibilizados

  • Shapefile
  • GeoJSON
  • TopoJSON (compacto, ideal para uso na web)
